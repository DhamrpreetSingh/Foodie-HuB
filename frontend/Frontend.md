## Frontend Purpose
Frontend provides complete user and admin web interface for:
- login/signup/guest access,
- menu browsing,
- cart and checkout,
- order tracking,
- profile management,
- support contact,
- and informational pages.

## Frontend Technologies
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons

## Frontend Structure
- `index.html` -> main landing page
- `pages/` -> app pages (home, menu, cart, checkout, orders, profile, about, contact)
- `Account/` -> login/admin panel UI pages
- `css/` -> custom styles
- `js/` -> frontend logic scripts
- `assets/` -> images/media resources
- `bootstrap/` -> local bootstrap library files
